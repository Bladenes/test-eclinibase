## **Réattribution du rendez-vous**

*\*N'oubliez pas de réattribuer le rendez-vous que vous venez d'annuler*

- 1- Ouvrez l'application eClinibase.
- 2- Ouvrez le module « Gestion des attentes ».
- 3- Recherchez l'usager ainsi que l'attente dans le service requis et désiré via la fenêtre de recherche.
- 4- Sélectionnez l'attente concernée.
- 5- Rechercher une disponibilité en appuyant sur le bouton dans la barre d'outils.
- 6- Octroyez le rendez-vous à partir de cette fenêtre.

Pour les utilisateurs qui doivent attribuer un rendez-vous sans avoir une attente, veuillez porter une attention particulière au détail de la fenêtre « Sélection d'attente » lorsque vous fixer un rendez-vous.

#### **2 Donner un rendez-vous**

Pour donner un rendez-vous à partir de l'attente, allez dans la bonne attente et appuyer sur "Rechercher la disponibilité liée à l'attente"

#### **7.3.6. Rendez-vous (de l'usager)**

Dans les rendez-vous (de l'usager), nous retrouvons tous les rendez-vous de l'usager sélectionné (effectués ou à venir). La restriction sur votre installation s'applique dans cette section.

Par défaut, la date du jour est inscrite dans la barre de recherche, vous ne verrez que les rendez-vous futurs. Aussi les rendez-vous annulés ne sont pas inclus

Donc, si vous cherchez un rendez-vous antérieur, n'oubliez pas d'enlever la date.

Pour consulter les rendez-vous, incluant les annulations, il vous suffit de descendre la barre de recherche et de cocher le statut « Annulé ».

Il est possible d'effectuer des recherches plus précises à l'aide des différents champs de la barre de recherche

### **7.4.3. Lien avec la recherche de disponibilité**

Une fois l'attente complétée, vous pouvez fixer un rendez-vous.

Appuyer sur « Rechercher la disponibilité liée à l'attente »

## **7.4.4. Recherche de disponibilité**

La fenêtre suivante s'ouvrira :

Dans la barre de recherche, tous les critères entrés dans votre attente seront automatiquement inscrits. Vous devez appuyer sur "Rechercher" pour faire apparaître les disponibilités.

La date de début de votre attente sera automatiquement transposée dans la case *À partir du* :. Cela ne vous empêchera pas de fixer un rendez-vous à une date antérieure, mais la case *Inclure les jours antérieurs* doit être cochée.

Il est possible de chercher une durée de rendez-vous précise avec le champ « Durée souhaitée du R-V » de la barre de recherche.

{26}------------------------------------------------

Une fois la recherche effectuée, vous serez en mesure de voir l'horaire de disponibilité de la ou des ressources, selon les critères inscrits dans la barre de recherche.

**Donc si une ressource n'apparait pas dans la liste, il est possible que celle-ci n'ait plus de disponibilités, le système vous avertira.** 

**Vous devrez donc générer de nouveaux horaires avant de fixer le rendezvous.** 

**Pour consulter les rendez-vous, vous pouvez utiliser la fenêtre gestion des rendez-vous, page de l'intervenant ou agenda.**

## Pour fixer le rendez-vous :

- Sélectionner la journée dans le calendrier central
- Sélectionner la bonne ressource et l'heure dans la fenêtre Détail de la disponibilité (double-cliquer sur l'heure).
- La fenêtre du rendez-vous s'ouvrira.
- Compléter les champs obligatoire (marqué d'un astérisque). Toutes les informations contenues dans l'attente seront automatiquement transposées dans le rendez-vous.

{27}------------------------------------------------

Une fois le rendez-vous fixé, appuyer sur OK et vous retournerez dans l'attente. Dans la section Fermeture de l'attente, vous remarquez qu'il y a maintenant une date de fermeture et la raison : Fin suite à un rendez-vous.

La date/heure d'affectation correspond à la date/heure du rendez-vous.

Vous devez ensuite vérifier si l'assignation pour cet usager est ouverte.

Pour plus de détail sur la recherche de disponibilité, consulter la section 7.5

{28}------------------------------------------------

#### **Fixer un rendez-vous sans horaire**

Il est possible que vous ayez le besoin de mettre un rendez-vous même s'il n'y a pas de disponibilité. C'est en général une solution exceptionnelle.

Vous devez partir de votre attente, aller dans la « Recherche de disponibilités » et effectuer la recherche.

Lorsque vous réaliser qu'il n'y a pas de disponibilité correspondante et que vous décidez de mettre le rendez-vous en surplus, appuyez sur « Prise de rendez-vous sans horaire »

La fenêtre pour fixer un rendez-vous apparaîtra, mais les informations de l'attente ne seront pas transcrites. Vous devrez les compléter.

{29}------------------------------------------------

- Commencer par l'installation, le service requis et la date et heure du rendez-vous.

- Suite à ces trois informations, une fenêtre de « Sélection d'une attente » ouvrira.

- Après avoir confirmé que l'attente proposée est bien celle pour laquelle vous voulez fixer un rendez-vous. (Vous pouvez aussi comparer le numéro de l'attente.)

- Sélectionner-la pour la lier avec les rendez-vous et appuyer sur Ok.

- Pour vous assurer que l'attente est bien liée au rendez-vous, un nouvel onglet dans le rendez-vous sera ajouté : Attente.

{30}------------------------------------------------

Simplement, compléter les champs de façon habituelle sans oublier de vérifier les cordonnées de l'usager. Il est possible de le faire en appuyant sur le petit bonhomme. Il s'agit d'un raccourci vers l'indexpatient.

Un rendez-vous ajouté de cette façon apparaitra toujours en rose dans les liste des rendez-vous.

{31}------------------------------------------------

### **7.5. Consultation des rendez-vous**

Pour consulter les rendez-vous qui sont déjà fixés, il y a deux façons :

- En utilisant la fenêtre Recherche de disponibilité (section 7.5.1)
- En utilisant la fenêtre Page de l'intervenant ou Agenda. (section 7.5.5 ou 7.5.6)

## **7.5.1. Recherche de disponibilité**

Pour consulter la recherche de disponibilité, aller dans l'onglet « eRendez-vous », choisir « Rendez-vous » et sélectionner « Recherche de disponibilité ».

Cette fenêtre est exactement la même que lorsqu'on appuie sur « Rechercher la disponibilité liée à l'attente ».

{36}------------------------------------------------

Dans la recherche de disponibilité, la recherche est possible seulement lorsqu'un horaire est généré. Si ce n'est pas le cas, la génération de l'horaire doit être effectuée.

Vous devez toujours fixer un rendez-vous à partir d'une attente.

{37}------------------------------------------------

#### **Détail des rendez-vous**

Il s'agit des rendez-vous qui sont attribués aux usagers. Le statut du rendez-vous est par défaut «Prévu». Il est de votre responsabilité d'effectuer la gestion des rendezvous.

Lorsque l'usager se présente, vérifiez ses informations, changez le statut pour présent, inscrivez la responsabilité de paiement et imprimez les documents nécessaires.

Rappel : Il est possible d'appliquer un modèle de disposition par défaut à toutes les listes du système.

{40}------------------------------------------------

### **7.5.2. Consulter le rendez-vous**

Lorsqu'on double-clic sur un rendez-vous, la fenêtre suivante s'ouvre :

C'est ici que l'on peut consulter les détails du rendez-vous, gérer la présence de l'usage ainsi que valider les coordonnées de l'usager.

{41}------------------------------------------------

#### **7.5.3. Transfert d'un rendez-vous**

Pour transférer un rendez-vous, il suffit de le sélectionner dans le détail des rendez-vous et d'appuyer sur transfert de rendez-vous.

Une nouvelle fenêtre de recherche de disponibilité s'ouvrira. Elle contient déjà tous les critères du rendez-vous à transférer. Appuyer sur « Rechercher » pour trouver toutes les disponibilités qui correspondent aux critères du rendez-vous. Lorsque la nouvelle plage est sélectionnée, appuyer sur « OK » pour arriver à la fenêtre de de transfert.

- Confirmer le changement avec l'usager
- Inscrire la raison du transfert (liste déroulante ou texte libre)
- Vérifier les coordonnées de l'usager
- Répéter les consignes.
- « OK »

La ressource aussi s'inscrit dans la barre de recherche, si l'usager peut voir une ressource ou une autre, il faut penser à l'enlever de la barre de recherche pour un maximum de disponibilités.

{43}------------------------------------------------

#### **7.5.4. Gestion des rendez-vous**

Lorsqu'on parle de gestion des rendez-vous, on parle de la présence, de la confirmation et de l'annulation des rendez-vous. La gestion des présences s'effectue au fur et à mesure que les usagers se présentent.

## **Confirmation**

Sortir la liste des rendez-vous à confirmer, aller dans l'onglet « eRendez-vous », choisir «Rendezvous» et sélectionner « Gestion des rendez-vous ».

- Choisir les dates des rendez-vous à confirmer. S'il s'agit d'une seule date, l'inscrire deux fois.
- Choisir le groupe de critère, ou tout autre critères selon les besoins.
- « OK »
- Téléphoner à chaque usager et changer le statut pour « Confirmer » ou téléphoner à chaque usager et changer le statut par lot :
  - Mettre un crochet vert à gauche
  - Appuyer sur « Confirmations »

Il est possible d'effectuer cette opération en lot pour les annulations (doivent avoir la même raison d'annulation).

Pour savoir comment ajouter ou enlever des colonnes de votre liste, allez voir l'**annexe 1**.

{44}------------------------------------------------

## **Présence**

Pour mettre une présence, il suffit de retourner dans le rendez-vous de l'usager et de cocher présent. La présence peut se mettre seulement à partir de la journée même du rendez-vous. Prenez toujours le temps de mettre à jour les informations et les coordonnées des usagers.

Appuyer sur «modification de l'identification» (Petit icône d'un pictogramme humain) afin d'accéder aux informations.

**Tous les index des Laurentides communiquent dans le but d'avoir les informations et les coordonnées de l'usager le plus à jour possible. Prenez le temps de les valider avec l'usager.**

C'est lorsqu'on met la présence que l'on doit créer le numéro de dossier local, si l'usager n'en a pas. Il suffit d'appuyer sur le dossier jaune (icône de dossier jaune).

La responsabilité de paiement est obligatoire au moment de la présence, puisque c'est à ce moment que vous avez l'opportunité de demander l'information à l'usager.

{45}------------------------------------------------

Lorsqu'on appuie sur « OK » la fenêtre d'impression s'ouvre. Pour la plupart des services, la carte de l'hôpital et le formulaire AH-280 sont sélectionnés par défaut pour l'impression. Il suffit de les décocher si on ne veut pas les imprimer. Les impressions sont paramétrées selon les services requis et peuvent donc différer.

Choisir les bonnes imprimantes et appuyer sur « OK »

Les imprimantes disponibles sont ceux paramétrées dans votre ordinateur.

{46}------------------------------------------------

#### **La réalisation du rendez-vous**

Lorsque le statut est utilisé, à l'enregistrement, le système vous demandera si vous voulez créer une attente de suivi.

Si vous répondez oui, une fenêtre d'attente s'ouvrira, déjà complétée. Il ne vous reste qu'à :

- Valider les informations
- Sélectionner le type de rendez-vous
- Sélectionner la priorité médicale de suivi
- Enregistrer.

L'usager est maintenant sur la liste d'attente.

## **L'annulation des rendez-vous**

À la fin d'une journée, il ne doit y avoir que rendez-vous au statut présent ou annulé. De même que pour les confirmations, il est possible d'effectuer les annulations en lot (doivent avoir la même raison d'annulation) en fin de journée. La raison principale est « Usager absent du RV\* ».

- Sélectionner les rendez-vous qui doivent être annulés :
  - Mettre un crochet vert à gauche
  - Appuyer sur « Annulations »

La fenêtre suivante vous permet de sélectionner la raison d'annulation et de choisir si vous souhaitez remettre l'usager sur la liste d'attente. Le choix de la raison et de la remise en attente sera valide pour tous les rendez-vous sélectionnés.

#### **7.5.5. Page de l'intervenant**

Pour consulter la page de l'intervenant, aller dans l'onglet « Module de base », choisir « Module de base » à nouveau et sélectionner « Page de l'intervenant ».

Votre ressource est automatiquement sélectionnée, donc votre horaire des rendez-vous apparait automatiquement. Votre compte Windows doit être liés à votre ressource (contacter les pilotes au besoin).

Le calendrier central est par défaut une visualisation à la journée, mais il est possible de voir la semaine de cinq ou sept jours ou même au mois

Les rendez-vous sont visuellement facile à identifier. Ils sont de différentes couleurs selon leur statut.

Les évènements sont toujours en bleus.

### **Transfert de rendez-vous**

Il est aussi possible de faire des transferts de rendez-vous dans la page de l'intervenant. La fonction est exactement la même que dans la fenêtre « Recherche de disponibilité »

 Sélectionner le rendez-vous (Un rendez-vous au statut « Présent » ne peut être transféré)

## **7.5.6. Agenda**

Pour consulter l'agenda, aller dans l'onglet « Agenda » et choisir « Agenda ».

Dans la barre de recherche, entrer le nom de la ressource pour laquelle vous souhaitez consulter l'horaire. Une fois la ressource sélectionnée, dans le calendrier plus haut, il est possible de savoir rapidement quand il y a des disponibilités à l'aide des traits de couleur.

Le reste des fonctions de l'agenda sont les même que l'on retrouve dans la page de l'intervenant.

## **7.6. eRendez-vous (Horaire)**

Pour avoir des disponibilités de rendez-vous, il doit y avoir des horaires générés. Ces générations d'horaire sont faites à partir de modèles d'horaire que l'on peut utiliser à chaque génération et ils sont modifiables. Il est important de comprendre que le modèle d'horaire doit refléter la réalité. Il est possible d'utiliser un modèle pour plusieurs ressources, si elles travaillent dans les mêmes services ou d'avoir plusieurs modèles pour une même ressource.

## **7.6.1. Création d'un modèle d'horaire**

Avant de créer un nouveau modèle d'horaire, toujours vérifier les modèles disponibles pour un groupe de critère, une ressource ou un service. Pour créer un nouveau modèle d'horaire appuyer sur le "+".

#### **Nombre de rendez-vous simultanés**

Il est possible d'augmenter le nombre de rendez-vous simultanés (en même temps). À utiliser lorsque les rendez-vous sont du même service requis et du même type de rendez-vous.

Cette option concerne la plage.

#### **Limite de rendez-vous**

Il est possible de créer des plages où il est possible de donner plusieurs rendez-vous différents (service requis ou type de rendez-vous) et de limiter à un certain nombre de rendez-vous.

Par exemple, dans la plage de 8h30 à 12h00, il est possible de mettre sept rendez-vous de nouveau cas mais puisque la limite est à deux, dès que le deuxième rendez-vous sera fixé, les autres disponibilités de nouveau cas disparaîtront entre 8h30 et 12h00.

Cette option est disponible lorsqu'on est positionné sur le service requis ou le type de rendezvous et concerne seulement la plage.

### **7.6.4. Génération d'un modèle d'horaire**

Avant de générer un horaire, le modèle doit absolument être enregistré. Ensuite, il faut appuyer sur le bouton « Générer les horaires » (Icône d'horloge).

Les champs qui doivent être complétés :

- ressource (s'il s'agit d'un modèle partagé entre plusieurs ressources) ;
- un commentaire à l'horaire, si pertinent (ce commentaire apparait dans une bulle dans la recherche de disponibilité) ;
- période (début et fin) ;
- type de génération ;
- les dates à générer (c'est dans cette fenêtre qu'on entre les dates pour la génération par date) ;
- la case « Approbation automatique de l'horaire à la génération » doit toujours être cochée ;
- appuyer sur le bouton « Générer » et fermer la fenêtre. Il est possible de désélectionner certains critères en décochant le crochet vert dans la partie du bas.

{64}------------------------------------------------

#### **7.6.5. Génération d'un horaire d'exception**

La génération d'un horaire d'exception se fait directement dans la recherche de disponibilité ou dans l'agenda. C'est à utiliser lorsqu'une ressource demande d'ouvrir un horaire pour une seule fois. Le modèle est donc inutile dans ce cas, puisqu'il ne sera plus utilisé (ne doit pas contenir de précision de la période).

Sélectionner la journée dans laquelle, la génération doit être faite et appuyer sur le bouton « Créer un horaire d'exception » (Icône d'horloge).

Compléter l'horaire d'exception selon les besoins et appuyer sur « OK » pour le générer.

**On ne peut pas mettre de précision de la période dans les horaires d'exception.**

{65}------------------------------------------------

## **7.6.6. Horaires et périodes**

C'est dans l'onglet « Horaires et périodes » qu'il sera possible de supprimer des horaires.

- Entrer les critères de recherche pour cibler ce qu'on désire supprimer.
- Sélectionner les plages dans la partie centrale.
- Sélectionner les périodes dans la partie de droite.
- Les actions permises sont dans le haut de la partie de droite.

**On ne peut pas supprimer ou modifier une période où il y a un rendez-vous.**
