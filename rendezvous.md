## **Réattribution du rendez-vous**

*\*N'oubliez pas de réattribuer le rendez-vous que vous venez d'annuler*

- 1- Ouvrez l'application eClinibase.
- 2- Ouvrez le module « Gestion des attentes ».
- 3- Recherchez l'usager ainsi que l'attente dans le service requis et désiré via la fenêtre de recherche.
- 4- Sélectionnez l'attente concernée.
- 5- Rechercher une disponibilité en appuyant sur le bouton dans la barre d'outils.
- 6- Octroyez le rendez-vous à partir de cette fenêtre.

#### **2 Donner un rendez-vous**

Pour donner un rendez-vous à partir de l'attente, allez dans la bonne attente et appuyer sur "Rechercher la disponibilité liée à l'attente"

#### **7.3.6. Rendez-vous (de l'usager)**

Donc, si vous cherchez un rendez-vous antérieur, n'oubliez pas d'enlever la date.

Pour consulter les rendez-vous, incluant les annulations, il vous suffit de descendre la barre de recherche et de cocher le statut « Annulé ».

Il est possible d'effectuer des recherches plus précises à l'aide des différents champs de la barre de recherche

### **7.4.3. Lien avec la recherche de disponibilité**

Une fois l'attente complétée, vous pouvez fixer un rendez-vous.

Appuyer sur « Rechercher la disponibilité liée à l'attente »

## **7.4.4. Recherche de disponibilité**

La fenêtre suivante s'ouvrira :

Il est possible de chercher une durée de rendez-vous précise avec le champ « Durée souhaitée du R-V » de la barre de recherche.

**Donc si une ressource n'apparait pas dans la liste, il est possible que celle-ci n'ait plus de disponibilités, le système vous avertira.**

**Vous devrez donc générer de nouveaux horaires avant de fixer le rendezvous.**

**Pour consulter les rendez-vous, vous pouvez utiliser la fenêtre gestion des rendez-vous, page de l'intervenant ou agenda.**

## Pour fixer le rendez-vous :

### Fixer un rendez-vous (résumé normé)
1) Depuis **Attente**, cliquer **Rechercher la disponibilité liée à l’attente**.  
2) Choisir **jour → ressource → heure** dans *Détail de la disponibilité*.  
3) Compléter les champs **obligatoires** (astérisque) ; l’attente se **transpose** automatiquement.

> **Pièges**  
> - Si une ressource n’apparaît pas : **plus de dispo** → **générer** de nouveaux horaires.  
> - Fixer **sans** disponibilité (cas exceptionnel) : les infos de l’attente **ne se transposent pas** ; les saisir manuellement (**vérifier l’onglet Attente** du R.V.).



- Sélectionner la journée dans le calendrier central
- Sélectionner la bonne ressource et l'heure dans la fenêtre Détail de la disponibilité (double-cliquer sur l'heure).
- La fenêtre du rendez-vous s'ouvrira.
- Compléter les champs obligatoire (marqué d'un astérisque). Toutes les informations contenues dans l'attente seront automatiquement transposées dans le rendez-vous.

La date/heure d'affectation correspond à la date/heure du rendez-vous.

Vous devez ensuite vérifier si l'assignation pour cet usager est ouverte.

Pour plus de détail sur la recherche de disponibilité, consulter la section 7.5

#### **Fixer un rendez-vous sans horaire**

Il est possible que vous ayez le besoin de mettre un rendez-vous même s'il n'y a pas de disponibilité. C'est en général une solution exceptionnelle.

Vous devez partir de votre attente, aller dans la « Recherche de disponibilités » et effectuer la recherche.

La fenêtre pour fixer un rendez-vous apparaîtra, mais les informations de l'attente ne seront pas transcrites. Vous devrez les compléter.

- Commencer par l'installation, le service requis et la date et heure du rendez-vous.

- Suite à ces trois informations, une fenêtre de « Sélection d'une attente » ouvrira.

- Après avoir confirmé que l'attente proposée est bien celle pour laquelle vous voulez fixer un rendez-vous. (Vous pouvez aussi comparer le numéro de l'attente.)

- Sélectionner-la pour la lier avec les rendez-vous et appuyer sur Ok.

- Pour vous assurer que l'attente est bien liée au rendez-vous, un nouvel onglet dans le rendez-vous sera ajouté : Attente.

Un rendez-vous ajouté de cette façon apparaitra toujours en rose dans les liste des rendez-vous.

### **7.5. Consultation des rendez-vous**

Pour consulter les rendez-vous qui sont déjà fixés, il y a deux façons :

- En utilisant la fenêtre Recherche de disponibilité (section 7.5.1)
- En utilisant la fenêtre Page de l'intervenant ou Agenda. (section 7.5.5 ou 7.5.6)

## **7.5.1. Recherche de disponibilité**

Cette fenêtre est exactement la même que lorsqu'on appuie sur « Rechercher la disponibilité liée à l'attente ».

Vous devez toujours fixer un rendez-vous à partir d'une attente.

#### **Détail des rendez-vous**

Rappel : Il est possible d'appliquer un modèle de disposition par défaut à toutes les listes du système.

### **7.5.2. Consulter le rendez-vous**

Lorsqu'on double-clic sur un rendez-vous, la fenêtre suivante s'ouvre :

C'est ici que l'on peut consulter les détails du rendez-vous, gérer la présence de l'usage ainsi que valider les coordonnées de l'usager.

#### **7.5.3. Transfert d'un rendez-vous**

Pour transférer un rendez-vous, il suffit de le sélectionner dans le détail des rendez-vous et d'appuyer sur transfert de rendez-vous.

- Confirmer le changement avec l'usager
- Inscrire la raison du transfert (liste déroulante ou texte libre)
- Vérifier les coordonnées de l'usager
- Répéter les consignes.
- « OK »

#### **7.5.4. Gestion des rendez-vous**



#### **La réalisation du rendez-vous**

Lorsque le statut est utilisé, à l'enregistrement, le système vous demandera si vous voulez créer une attente de suivi.

Si vous répondez oui, une fenêtre d'attente s'ouvrira, déjà complétée. Il ne vous reste qu'à :

- Valider les informations
- Sélectionner le type de rendez-vous
- Sélectionner la priorité médicale de suivi
- Enregistrer.

L'usager est maintenant sur la liste d'attente.

## **L'annulation des rendez-vous**

- Sélectionner les rendez-vous qui doivent être annulés :
  - Mettre un crochet vert à gauche
  - Appuyer sur « Annulations »

#### **7.5.5. Page de l'intervenant**

Le calendrier central est par défaut une visualisation à la journée, mais il est possible de voir la semaine de cinq ou sept jours ou même au mois

Les rendez-vous sont visuellement facile à identifier. Ils sont de différentes couleurs selon leur statut.

Les évènements sont toujours en bleus.

### **Transfert de rendez-vous**

Sélectionner le rendez-vous (Un rendez-vous au statut « Présent » ne peut être transféré)

## **7.5.6. Agenda**

Pour consulter l'agenda, aller dans l'onglet « Agenda » et choisir « Agenda ».

Le reste des fonctions de l'agenda sont les même que l'on retrouve dans la page de l'intervenant.

## **7.6. eRendez-vous (Horaire)**



## **7.6.1. Création d'un modèle d'horaire**



#### **Nombre de rendez-vous simultanés**

Cette option concerne la plage.

#### **Limite de rendez-vous**

Cette option est disponible lorsqu'on est positionné sur le service requis ou le type de rendezvous et concerne seulement la plage.

### **7.6.4. Génération d'un modèle d'horaire**

Les champs qui doivent être complétés :

- ressource (s'il s'agit d'un modèle partagé entre plusieurs ressources) ;
- un commentaire à l'horaire, si pertinent (ce commentaire apparait dans une bulle dans la recherche de disponibilité) ;
- période (début et fin) ;
- type de génération ;
- les dates à générer (c'est dans cette fenêtre qu'on entre les dates pour la génération par date) ;
- la case « Approbation automatique de l'horaire à la génération » doit toujours être cochée ;
- appuyer sur le bouton « Générer » et fermer la fenêtre. Il est possible de désélectionner certains critères en décochant le crochet vert dans la partie du bas.

#### **7.6.5. Génération d'un horaire d'exception**

Sélectionner la journée dans laquelle, la génération doit être faite et appuyer sur le bouton « Créer un horaire d'exception » (Icône d'horloge).

Compléter l'horaire d'exception selon les besoins et appuyer sur « OK » pour le générer.

**On ne peut pas mettre de précision de la période dans les horaires d'exception.**

## **7.6.6. Horaires et périodes**

C'est dans l'onglet « Horaires et périodes » qu'il sera possible de supprimer des horaires.

- Entrer les critères de recherche pour cibler ce qu'on désire supprimer.
- Sélectionner les plages dans la partie centrale.
- Sélectionner les périodes dans la partie de droite.
- Les actions permises sont dans le haut de la partie de droite.

**On ne peut pas supprimer ou modifier une période où il y a un rendez-vous.**
