# eClinibase — Site GitHub Pages (Copilote)

## Structure
- `/index.md` — Sommaire
- `/attentes.md` — Gestion des attentes (Détail, Communication, Mécanisme d'accès, etc.)
- `/rendezvous.md` — Recherche de disponibilité, gestion des RDV, Agenda, eRendez-vous
- `/normes-saisie.md` — Normes de saisie (incluant priorités médicales)
- `/procedures.md` — Marche à suivre, rôles, accès, extraction, annexes

## Publication
1. Copiez **tous les fichiers** à la racine de votre dépôt GitHub Pages.
2. Activez GitHub Pages depuis _Settings → Pages_ (Source = `main`).
3. Vos URLs seront :
   - `/attentes/`
   - `/rendezvous/`
   - `/normes-saisie/`
   - `/procedures/`

## Remarques
- Les titres/sections originaux ont été conservés et répartis par thème.
- Aucune information n’a été supprimée ; si une section était ambiguë, elle a été versée dans **Procédures**.
